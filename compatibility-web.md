## Web 产品兼容范围

#### PC端
* 操作系统
  * Windows NT 5.1+ （XP 系统的用户量依然很庞大）
  * Mac OS X Edge （作为非 Win 系统的标准环境做兼容）
* 浏览器
  * IE 9+ （目前国内的主流 IE 版本）
  * Chrome 26+ （目前还存在用户群的最低版本）
  * Firefox Edge （Firefox 用户总是会升级到最新版）
  * 国产浏览器 `360高速` / `搜狗极速` / `QQ浏览器`

#### 移动端
* 操作系统
  * IOS 7+
  * Android 4+ 
* 浏览器
  * 国产浏览器 `UC` / `QQ`
  * 其它重要平台 `微信嵌入`
